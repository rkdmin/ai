# 📸 촬영 가이드

정확한 분석을 위해 아래 가이드에 맞춰 사진을 찍어주세요.
촬영 각도·거리에 따라 분석 정확도가 크게 달라집니다.

---

## 👗 의상

- **민무늬 검정 민소매** 착용 (어깨·쇄골이 보여야 함)
- 패턴·로고·색상 있는 옷 X

---

## 🖼 배경 & 조명

- **민무늬 밝은 배경** (살구색 제외)
- 배경에 그림자가 지지 않도록 주의
- 조명은 **앞쪽**에 배치 (역광 X)

---

## 📱 카메라

- **후면 카메라 / 기본 모드** 사용
- **광각(0.5×) 금지** — 얼굴 왜곡 발생
- 갤럭시 울트라 기종 사용 금지
- 카메라를 기울이지 말고 **수직** 유지

---

## 🧍 촬영 자세

| 항목 | 기준 |
|------|------|
| **헤어** | **앞머리까지 완전히 올려서 포니테일로 묶기** |
| 거리 | 약 60cm (팔 길이) |
| 렌즈 위치 | 눈과 입술 사이 |
| 자세 | 차렷 / 무표정 |
| 프레임 | 어깨 양옆에 여백 나오게 |

---

## ⚠️ 자주 하는 실수

- 앞머리를 내린 채 촬영 → 이마·얼굴형 분석 불가
- 광각 카메라 사용 → 얼굴 비율 왜곡
- 측면·반측면 촬영 → 정면 분석 불가
- 마스크·선글라스 착용 → 이목구비 분석 불가
- 어두운 배경 또는 역광 → 윤곽 인식 오류

---

## ✅ 체크리스트

- [ ] 검정 민소매 착용

- [ ] 밝은 민무늬 배경
- [ ] 후면 기본 카메라 (광각 X)
- [ ] 카메라 수직 / 거리 60cm
- [ ] 정면 / 무표정 / 차렷 자세
