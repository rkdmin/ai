# 📸 촬영 가이드

정확한 분석을 위해 아래 가이드에 맞춰 사진을 찍어주세요.
촬영 각도·거리에 따라 분석 정확도가 크게 달라집니다.

---

## 📌 공통 사항 (모든 사진 동일 적용)

### 👗 의상
- **민무늬 검정 민소매** 착용 (어깨·쇄골이 보여야 함)
- 패턴·로고·색상 있는 옷 X

### 💇 헤어
- **앞머리까지 완전히 올려서 포니테일로 묶기**

### 🖼 배경 & 조명
- **민무늬 밝은 배경** (살구색 제외)
- 배경에 그림자가 지지 않도록 주의
- 조명은 **앞쪽**에 배치 (역광 X)

### 📱 카메라
- **후면 카메라 / 기본 모드** 사용
- **광각(0.5×) 금지** — 얼굴 왜곡 발생
- 갤럭시 울트라 기종 사용 금지
- 카메라를 기울이지 말고 **수직** 유지

### 📐 거리 & 프레임
- 거리: 약 60cm (팔 길이)
- 렌즈 위치: 눈과 입술 사이
- 프레임: 어깨 양옆에 여백 나오게
- 자세: 차렷 / 무표정

---

## 📷 촬영 방법

### ✅ 정면 (필수, v1.0은 1장만)
- 카메라를 정면으로 바라보기
- 고개를 앞뒤로 기울이지 말 것

> 측면(90도·45도)은 v1.x 검토. v1.0은 정면 1장만 받음 (`docs/phase1-quality.md` 1-3 참조).

---

## ⚠️ 자주 하는 실수

- 앞머리를 내린 채 촬영 → 이마·얼굴형 분석 불가
- 광각 카메라 사용 → 얼굴 비율 왜곡
- 마스크·선글라스 착용 → 이목구비 분석 불가
- 어두운 배경 또는 역광 → 윤곽 인식 오류

---

## ✅ 사용자 직접 촬영 체크리스트

> 아래 항목은 촬영 전에 **네가 직접 확인**해야 한다.

- [ ] 검정 민소매 착용
- [ ] 앞머리까지 포니테일로 올려 묶음
- [ ] 밝은 민무늬 배경
- [ ] 후면 기본 카메라 (광각 X)
- [ ] 카메라 수직 / 거리 60cm
- [ ] **정면** 사진 — 무표정 / 차렷 / 어깨 여백
