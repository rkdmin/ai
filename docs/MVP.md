# 🎯 MVP — Android 출시 기준

> 이 문서에 있는 것이 전부 완료되어야 출시 가능.
> 하나라도 미완이면 출시 불가.

---

## 출시 전 필수 완료 항목

### 핵심 기능
- [ ] 정면 사진 업로드 + 측면 사진 선택 업로드
- [ ] MediaPipe(백엔드) + Claude Vision 얼굴형 분석
- [ ] 퍼스널컬러 확정 흐름 (알면 선택 / 모르면 3문항)
- [ ] RAG 기반 헤어 / 메이크업 / 종합 카드 4장 생성
- [ ] 카드 상세 + 전문가 코멘트

### 보안
- [ ] API 키 백엔드로 이동 (클라이언트 노출 없음)
- [ ] Rate Limiting (비로그인 / 로그인 차등)

### 인증
- [ ] 카카오 로그인
- [ ] 구글 로그인
- [ ] 게스트 1회 체험

### 히스토리
- [ ] 분석 결과 저장 (로그인 시 자동)
- [ ] 최근 5회 조회

### 수익화
- [ ] 카드 잠금 구조: Rank 3 무료, Rank 1·2는 보상형 광고 시청 후 해금
- [ ] 보상형 광고 (AdMob Rewarded) — 광고 1회 시청 → 카드 1장 해금
- [ ] Google AdMob 배너 광고
- [ ] 유료 플랜 인앱결제 (광고 없이 전체 카드 + Gemini 사진 생성)
- [ ] 쿠팡 파트너스 — 카드 상세 내 "내 퍼스널컬러 추천 TOP 3" 상품 형태

### 바이럴
- [ ] 결과 카드 이미지 저장/공유 (react-native-view-shot)
- [ ] 감성 얼굴형 레이블 (카드 목록 최상단 표시)
- [ ] 연예인 스타일 매칭 텍스트 (RAG 데이터 필드 추가)
- [ ] 전후 비교 토글 (원본 사진 ↔ Gemini 적용 사진, 카드 상세)

### 뷰티 트렌드 피드
- [ ] 하단 탭바 구조: 홈(분석) / 트렌드 / 히스토리
- [ ] 주 1회 RSS 수집 (뷰티누리 + 얼루어코리아 + 네이버 뷰티 뉴스 API)
- [ ] Gemini로 기사 요약 + 얼굴형/퍼스널컬러 태깅 → Supabase 저장
- [ ] 트렌드 탭: 내 얼굴형·퍼컬 태그 기사만 필터링해서 표시
- [ ] 원문 링크 제공 (저작권: 요약만 표시, 원문은 외부 링크)

### 앱
- [ ] React Native (Expo) 빌드
- [ ] Android 실기기 테스트 완료
- [ ] 앱 아이콘 / 스플래시 스크린
- [ ] 개인정보처리방침 페이지

## 기술 스택

| 영역 | 기술 |
|------|------|
| 앱 | React Native (Expo) |
| 백엔드 | Python + FastAPI |
| 얼굴 측정 | MediaPipe (Python, 백엔드) |
| 얼굴 분석 + 카드 생성 | Gemini 2.5 Flash |
| 이미지 생성 | Gemini 2.5 Flash (이미지 생성 모드, 유료 전용) |
| DB / 인증 | Supabase |
| 앱 빌드 | Expo EAS Build |
| 인앱결제 | RevenueCat |
| 광고 | Google AdMob |
| 제휴 | 쿠팡 파트너스 |
